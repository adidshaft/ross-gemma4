# ROSS-Gemma4 Demo Script

This script outlines the video demo flow for the Gemma 4 Good Hackathon.

1. **Launch App:** 
   Open ROSS-Gemma4 on an iOS device or simulator.

2. **Model Download Simulation:** 
   Show the UI requesting the user to download the Gemma 4 capability pack. (Wait for simulated download completion or bypass it via Demo Mode).

3. **Workspace Intake:** 
   Navigate to the local workspace and import a sample case file (e.g., a PDF document).

4. **Local Analysis:** 
   Ask Ross a question about the document (e.g., "What are the operative directions in this order?").

5. **View Demo Label:** 
   Highlight the UI label showing "Demo Mode — model response simulated for walkthrough." to prove transparency.

6. **Chronology/Issue Extraction:** 
   Trigger a local document extraction and view the generated chronology candidates.

7. **Public Law Search (Local Shaping):** 
   Demonstrate the query shaping for public law search, proving that only sanitized queries are sent externally and all document context remains on the device.
