# Offline Behavior

## Works offline

- Create case
- Import PDF
- Quick capture
- OCR where on-device capability exists
- Organize documents
- Keyword search
- View prior exports
- Run local workflows when the required pack is installed

## Degraded offline behavior

- If no pack is installed, the app remains usable for capture, organization, and extractive workflows only
- If Quick Start is installed, the app can support Instant Mode and small local tasks
- Public-law search is unavailable offline
- Entitlement refresh and model catalog checks wait until connectivity returns

## User messaging

- Explain what still works
- Explain what is waiting
- State clearly that case files remain local and unaffected
