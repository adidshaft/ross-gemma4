# ROSS-Gemma4 Demo Script

**Setup**
1. Open ROSS-Gemma4. Navigate to Settings -> Assistant.
2. Tap "Install Quick Associate" to download the Gemma 4 E2B Q4 capability pack.

**Import**
1. Create a new matter "State v. Doe".
2. Import three sample PDF statements into the case folder.

**Analyze**
1. Tap "Ask ROSS". 
2. Ask, "What are the timeline discrepancies between Witness A and Witness B?"
3. The app retrieves context and generates a local response in seconds.

**Draft**
1. Select the highlighted discrepancies and tap "Draft Memo".
2. ROSS-Gemma4 writes a formal memo detailing the contradictions. 
3. Tap "Save to Case Files".
